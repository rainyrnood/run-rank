# 런등수 (Run Rank)

공개된 일반인 러닝 통계 기반의 가상 집단 안에서, 달리는 중 **실시간으로 내 등수**와 바로 앞/뒤 주자와의 거리(m)를 보여주는 러닝 PWA 프로토타입.

## 사용
- 아이폰 사파리에서 Pages URL 접속 → 공유 → "홈 화면에 추가"
- **시뮬레이션 모드**: 페이스/배속 슬라이더로 책상에서 동작 확인
- **GPS 실측 모드**: 밖에서 시작 → 실시간 등수 갱신 (HTTPS 필요, 화면 켜둔 상태)

## 구조
- `index.html` — 앱 전체(계산·UI 모두 클라이언트), `manifest.webmanifest`/`sw.js`/`icon.svg` — PWA 구성
- 백엔드 없음. 모든 계산은 폰에서 수행.

> 가상 집단 페이스 분포는 임시 placeholder(아마추어 추정치). 추후 실제 공개통계로 교체 예정.
